Этот репозиторий содержит проект Канбан Доску, предназначенный для управления задачами и проектами с использованием метода Канбан. Доска позволяет пользователям визуализировать рабочий процесс, отслеживать выполнение задач и управлять очередями задач.

Основные функции:
Создание задач: Пользователи могут добавлять новые задачи с описанием и цветом 
Колонки задач: Доска включает три основные колонки:
Надо сделать: Задачи, которые требуют выполнения.
В процессе: Задачи, над которыми в данный момент работает команда.
Сделано: Завершенные задачи.
Архив: Удобная функция для перемещения завершённых задач в архив, чтобы они не загромождали основную доску, но при этом оставались доступными для просмотра.
Удаление задач: Возможность массового удаления задач на доске при необходимости, что помогает начать с чистого листа.
Визуализация прогресса: Интерфейс предоставляет наглядное представление текущего состояния задач и позволяет командам быстро оценивать прогресс.
Этот проект стремится помочь командам эффективно управлять своими задачами, улучшая их организацию и видимость, а также обеспечивая гибкость в управлении рабочими процессами.
